# FAQ v3.1.0 (2026)

## 🤖 AI

### Q: Apakah AI mengirim data ke server eksternal?
**A**: Tidak. Semua AI processing dilakukan secara lokal di browser. Data tidak dikirim ke server AI eksternal.

### Q: Seberapa akurat AI auto-tagging?
**A**: Akurasi rata-rata 94.5% untuk tagging. Confidence score ditampilkan di setiap tag.

### Q: Apakah AI bisa belajar dari koreksi user?
**A**: Saat ini AI menggunakan rule-based system. Machine learning adaptif direncanakan untuk v3.2.0.

## 🔗 Blockchain

### Q: Apakah blockchain memperlambat aplikasi?
**A**: Mining time rata-rata 100-500ms per block. Tidak signifikan untuk penggunaan normal.

### Q: Bisakah blockchain di-reset?
**A**: Ya, admin dapat mereset blockchain dari pengaturan. Backup otomatis dibuat sebelum reset.

## 🔐 Biometric

### Q: Apakah data sidik jari disimpan?
**A**: Tidak. Hanya public key yang disimpan. Data biometrik mentah TIDAK PERNAH disimpan.

### Q: Bagaimana jika device biometric rusak?
**A**: Password login selalu tersedia sebagai fallback.

## 🎤 Voice

### Q: Bahasa apa yang didukung?
**A**: Bahasa Indonesia dan English. Bahasa daerah dalam pengembangan.

### Q: Apakah voice command bekerja offline?
**A**: Speech recognition memerlukan koneksi internet. Command execution tetap bekerja offline.

## 📱 PWA

### Q: Apakah v3.1.0 mendukung Window Controls Overlay?
**A**: Ya! Install PWA di desktop untuk native title bar experience.

### Q: Berapa ukuran instalasi PWA?
**A**: ~5MB initial, ~15MB dengan cache penuh (termasuk AI models).
