# Changelog

All notable changes to this project will be documented in this file.

## [3.1.0] - 2026-07-10

### 🚀 MAJOR FEATURES
#### 🤖 AI Engine
- AI Smart Search dengan NLP & semantic understanding
- AI Auto-Tagging untuk klasifikasi surat otomatis
- AI Anomaly Detection untuk deteksi surat mencurigakan
- AI Predictive Analytics untuk prediksi tren surat
- AI Document Assistant untuk bantuan penulisan

#### 🔗 Blockchain Audit Trail
- Immutable audit log dengan SHA-256 hash chain
- Proof of Work mining untuk verifikasi integritas
- Chain verification real-time
- Anti-tampering detection

#### 🔐 Biometric Authentication
- WebAuthn/FIDO2 standard support
- Face ID authentication (Windows Hello, Apple Face ID)
- Fingerprint authentication (Touch ID, Android Biometric)
- Platform authenticator integration

#### 🎤 Voice Commands
- Speech recognition untuk navigasi
- Voice search dengan multi-language
- Custom voice commands
- Keyboard shortcut (Ctrl+Shift+V)

#### 🌐 Multi-Language (i18n)
- Bahasa Indonesia full support
- English full support
- Dynamic language switching (Ctrl+L)
- Persistent locale storage

#### 🌙 Dark Mode
- Full dark theme untuk seluruh aplikasi
- Auto-switch mengikuti sistem
- Manual toggle (Ctrl+D)
- Persistent theme storage

#### 📱 PWA Enhancements
- Window Controls Overlay (WCO) support
- File Handler API integration
- Share Target API v2
- App Shortcuts dengan AI actions
- Periodic Background Sync (setiap 30 menit)
- Push Notification Actions (quick approve/reject)

### 🔧 IMPROVEMENTS
- Upgrade ke Material Design 3
- Optimasi performa dengan lazy loading
- Enhanced Service Worker v3.1.0
- Improved offline experience dengan AI lokal
- Better error handling dengan AI suggestions
- Enhanced security dengan WebAuthn

### 🐛 BUG FIXES
- Fix memory leak pada long-running sessions
- Fix race condition pada background sync
- Fix cache invalidation issues
- Fix responsive layout pada tablet
- Fix notification duplicate issue
- Fix file upload progress tracking

### ⚠️ BREAKING CHANGES
- API v2 deprecated (gunakan API v3)
- Minimum browser requirement updated
- Service Worker cache structure changed
- LocalStorage schema updated

### 📦 DEPENDENCIES
- Added: fuse.js v7.0.0 (AI search)
- Added: crypto-js v4.2.0 (blockchain)
- Added: i18next v23.0.0 (multi-language)
- Added: workbox v7.0.0 (PWA)
- Updated: chart.js v4.4.0
- Updated: vite v5.0.0
