# Panduan Pengguna v3.1.0 (2026)

## 🤖 Fitur AI Baru

### AI Smart Search
Gunakan search bar dengan icon 🤖 untuk pencarian pintar:
- Ketik kata kunci alami seperti "surat dari DPRD bulan lalu"
- AI akan memahami konteks dan memberikan hasil relevan
- Hasil diurutkan berdasarkan skor relevansi AI

### AI Auto-Tagging
Saat membuat surat, AI otomatis menambahkan tag:
- Tag muncul setelah menyimpan surat
- Klik tag untuk filter surat sejenis
- Confidence score menunjukkan tingkat kepercayaan AI

### AI Anomaly Detection
AI mendeteksi surat mencurigakan:
- Notifikasi muncul di dashboard
- Klik untuk lihat detail anomali
- Score > 50 perlu perhatian khusus

### AI Predictive Analytics
Prediksi tren surat di dashboard:
- Grafik prediksi dengan confidence band
- Rekomendasi AI untuk tindakan
- Update otomatis setiap minggu

## 🔗 Blockchain Audit Trail

### Melihat Blockchain
1. Buka **Audit Log**
2. Klik tab **Blockchain**
3. Lihat chain dengan hash terverifikasi

### Verifikasi Integritas
- Klik **Verify Chain** untuk cek integritas
- Hash yang cocok = data tidak diubah
- Hash berbeda = ada percobaan manipulasi

## 🔐 Biometric Login

### Setup Biometric
1. Buka **Pengaturan > Keamanan**
2. Klik **Daftarkan Biometric**
3. Ikuti petunjuk Face ID/Fingerprint

### Login dengan Biometric
1. Di halaman login, klik **🔐 Login dengan Biometric**
2. Verifikasi wajah/sidik jari
3. Otomatis masuk ke dashboard

## 🎤 Voice Commands

### Mengaktifkan
- Tekan **Ctrl+Shift+V** untuk mulai
- Atau klik tombol 🎤 di header

### Perintah Suara
| Perintah | Aksi |
|----------|------|
| "Buka dashboard" | Navigasi ke dashboard |
| "Surat masuk" | Buka daftar surat masuk |
| "Surat keluar" | Buka daftar surat keluar |
| "Approval" | Buka halaman approval |
| "Cari [kata kunci]" | Fokus ke search bar |
| "Dark mode" | Aktifkan dark mode |
| "Light mode" | Aktifkan light mode |

## 🌙 Dark Mode

### Aktivasi
- **Otomatis**: Ikuti pengaturan sistem
- **Manual**: Tekan **Ctrl+D**
- **Toggle**: Klik 🌙 di header

## 🌐 Multi-Language

### Ganti Bahasa
- Tekan **Ctrl+L** untuk switch
- Atau klik **ID/EN** di header
- Bahasa disimpan otomatis
