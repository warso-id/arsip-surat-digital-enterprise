Sistem Manajemen Arsip Surat Digital Enterprise dengan fitur **AI**, **Blockchain**, dan **Biometric Authentication**.

## ✨ Fitur Utama

### 🤖 Artificial Intelligence
- Smart Search dengan NLP
- Auto-tagging dokumen
- Anomaly detection
- Predictive analytics
- Document analysis

### ⛓️ Blockchain
- Immutable audit trail
- Proof-of-Work consensus
- Real-time chain verification
- Transparent log system

### 🖐️ Biometric Authentication
- WebAuthn/FIDO2 support
- Fingerprint recognition
- Face recognition (device-dependent)
- Multi-factor authentication

### 📱 Progressive Web App (PWA)
- Installable on all platforms
- Offline support
- Push notifications
- Background sync
- Cross-platform (Windows, iOS, Android, macOS, Linux)

### 📊 Fitur Manajemen Surat
- Surat Masuk & Keluar
- Disposisi elektronik
- Approval workflow
- QR Code generation
- Template surat
- Export ke Excel/PDF
- Backup & Restore
