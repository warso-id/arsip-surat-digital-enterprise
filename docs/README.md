## 📋 Deskripsi

**Arsip Surat Digital Enterprise** adalah sistem manajemen arsip surat berbasis web yang menggunakan **Google Spreadsheet** sebagai database dan **Google Apps Script (Code.gs)** sebagai backend API.

### 🔗 Teknologi
| Layer | Teknologi |
|-------|-----------|
| **Frontend** | HTML5, CSS3, JavaScript (Vanilla) |
| **Backend** | Google Apps Script (Code.gs) |
| **Database** | Google Spreadsheet |
| **Auth** | JWT + Google Account |
| **Deploy** | GitHub Pages |
