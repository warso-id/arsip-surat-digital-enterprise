## Cara Berkontribusi

1. **Laporkan Bug**
   - Gunakan template bug report
   - Sertakan langkah-langkah reproduksi
   - Lampirkan screenshot jika memungkinkan

2. **Usulkan Fitur**
   - Gunakan template feature request
   - Jelaskan use case
   - Diskusikan di issues terlebih dahulu

3. **Kirim Kode**
   - Fork repository
   - Buat branch fitur
   - Ikuti coding standards
   - Tulis test
   - Update dokumentasi
   - Buat Pull Request
